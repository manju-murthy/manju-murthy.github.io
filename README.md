## Local Development

1. **Clone the repository**:
   ```bash
   git clone https://github.com/manju-murthy/manju-murthy.github.io.git
   cd manju-murthy.github.io
   ```

2. **Serve locally**:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Open in browser**:
   Navigate to `http://localhost:8000`

## Deployment

This portfolio is designed for GitHub Pages deployment:

1. Push to a repository named `manju-murthy.github.io`
2. Enable GitHub Pages in repository settings
3. Site will be available at `https://manju-murthy.github.io/`

### Custom Domain (Optional)
To use a custom domain:
1. Add `CNAME` file with your domain
2. Configure DNS with your domain provider
3. Enable HTTPS in GitHub Pages settings
